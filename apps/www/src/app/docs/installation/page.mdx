# Installation

Getting started with Nen is incredibly easy. We provide a CLI tool to scaffold everything you need, or you can install the packages manually.

## Using the CLI (Recommended)

The easiest way to start using Nen is to scaffold a new Next.js project using our CLI tool. It automatically configures Tailwind CSS, Shadcn UI, and sets up the Nen server and client templates for you.

```bash
npx create-nen-app@latest my-secure-app
```

Once the setup is complete, navigate to your project and start the development server:

```bash
cd my-secure-app
npm run dev
```

## Manual Installation

If you want to integrate Nen into an existing project, you can install the client and server packages via npm.

```bash
npm install @nen/client @nen/server
```

### Prerequisites
- Node.js 18 or later
- React 18 or later (for client-side usage)
- A modern browser that supports WebAssembly (for the client engine)

Next, head over to the [Usage](/docs/usage) guide to see how to initialize the Nen engine in your application.
