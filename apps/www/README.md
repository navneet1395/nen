# Isogeny Product Website

The official marketing and documentation website for Isogeny.

## Features

- Built with Next.js 15 App Router.
- Styled with Tailwind CSS v4.
- High-end aesthetics and micro-animations via Framer Motion.
- Comprehensive Markdown-like documentation layout via `@tailwindcss/typography`.

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3001](http://localhost:3001) with your browser to see the landing page and documentation.
